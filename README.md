# 🇺🇸 Supermarket Sales Analysis

## 📊 Project Overview

This project analyzes supermarket sales data using SQL in PostgreSQL.

The objective is to explore the dataset and extract business insights such as 
- revenue performance
- customer behavior
- product trends

## ❓ Business Questions

1. Which city generates the most revenue?
2. Which product line sells the most items?
3. What is the most used payment method?
4. What is the average transaction value?

# 🇧🇷 Análise de Vendas de Supermercado

## 📊 Visão do Projeto

Este projeto analisa dados de vendas de um supermercado utilizando SQL no PostgreSQL.

O objetivo é explorar o dataset e extrair insights de negócio como:

- desempenho de vendas
- comportamento dos clientes
- categorias de produtos mais vendidas

## ❓ Perguntas de Negócio

1. Qual cidade gera mais receita?
2. Qual linha de produto vende mais?
3. Qual forma de pagamento é mais utilizada?
4. Qual é o ticket médio das vendas?
